# 开发指南

## 前置知识

- React
- Electron
- Typescript

## 项目初始化

```bash

$ git clone xx
$ cd labelbee-client
$ npm install
$ npm run start # 启动 web 项目
$ npm run electron-dev # 启动客户端项目
```

## 打包

```bash
$ npm run packager:win # 创建 window 版本客户端
$ npm run packager:mac # 创建 mac 客户端
$ npm run packager:linux # 创建 linux 客户端
```
